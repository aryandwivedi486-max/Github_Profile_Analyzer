## About This Project

This is a small learning project created while exploring REST APIs and API integration in Python.

After learning API fundamentals such as requests, responses, JSON, endpoints, authentication, and working with real APIs, I wanted to build a simple project to understand how APIs are used in practice.

The GitHub Profile Analyzer uses the GitHub REST API to fetch public profile information and display useful details such as username, followers, following count, public repositories, account creation date, and profile URL.

This project helped me gain hands-on experience with:

* REST APIs
* API Requests and Responses
* JSON Data Handling
* Python Programming
* Error Handling
* Working with Real-World APIs

This project marks my first step from learning API concepts to implementing them in a practical application.
